# Design Decisions

## 4/3/26
- STM32 development board will be manually placed onto the PCB, which will have
female pin header connectors for it to hook up to
 - Will be using morpho connectors vs. Arduino connectors for more flexibility

## 4/4/26
- Datasheets will be stored in the git files instead of Notion, as notion does
not have an easy way to import PDFs
- Using wget to get PDFs and other files into the repo/working directory
